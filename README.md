# 🔐 Test d'intrusion - Metasploit 6 (Mini-réseau virtualisé)

Démarche complète d’un test d’intrusion réalisée sur un mini-réseau simulant une infrastructure d’entreprise, composé de Windows Server 2008 et Ubuntu 14.04.

Ce projet a été réalisé dans le cadre de la Licence 3 Informatique (IED), sous la direction de Monsieur Philippe Kislin, Enseignant-Chercheur à l’Université Paris 8 Vincennes.
